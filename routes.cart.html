def view_cart(): (source) ¶

Handle the view cart request. This function checks if the current user is authenticated. If not, it returns a 401 status code with a message indicating that the user is not logged in. If the user is authenticated, it filters the parameters and renders the 'cart.html' template with the filtered parameters.

Returns:
tuple: A message and a status code if the user is not authenticated. str: Rendered 'cart.html' template with filtered parameters if the user is authenticated.
